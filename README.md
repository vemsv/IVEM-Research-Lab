# IVEM Research Lab 🧠✨

## 📌 연구소 개요
**IVEM Research Lab**은 인간과 AI의 협업 방식에 대한 연구를 진행하는 연구소입니다.  
우리는 AI의 사고 과정, 메모리 최적화, 협업 모델, 감각적 반응 등을 연구하며,  
인간과 AI가 함께 발전할 수 있는 환경을 만들어갑니다.
